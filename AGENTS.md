
## Testing
- Use JUnit 6 for all tests.
- Make extensive use of mocking where practical to isolate unit behavior.
- Every generated module must include at least basic test coverage.

## Java Conventions
- Reduce boilerplate code with Lombok where appropriate.
- Prefer Java `record` types over Lombok `@Data`.
- All classes should be annotated with `@NullMarked` by default.
- All classes should be annotated with `@Slf4j` by default.
